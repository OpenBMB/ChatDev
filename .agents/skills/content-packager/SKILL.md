# Content Packager

Use this skill when the task is no longer about discovering facts, but about shaping them into audience-ready content.

This skill is intended for:
- turning a research brief into a post, article, or script
- improving hooks, pacing, and readability
- adapting the same core material for different audiences

Workflow:
1. Activate this skill after the core facts or draft already exist.
2. Identify the target audience, desired tone, and expected output format.
3. Reframe the material around:
   - strongest hook
   - clean structure
   - memorable takeaways
   - explicit next step or CTA
4. Keep edits audience-aware instead of merely making the text longer.

Rules:
1. Do not invent facts to make the copy stronger.
2. Preserve any hard constraints or approved claims from earlier stages.
3. Optimize for clarity and momentum, not just polish.

Expected output shape:
- Audience
- Hook
- Final packaged content
- Optional variants
